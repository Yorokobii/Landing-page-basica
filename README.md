# Portafolio Freelancer — Luis Carlos Alegria
Sitio web personal tipo portafolio desarrollado como práctica de HTML y CSS.

## 📋 Descripción
Página web de una sola página (landing page) que presenta los servicios de un diseñador
y desarrollador web freelancer. Incluye navegación, sección hero, servicios y formulario de contacto.

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Google Fonts (Krub)
- Normalize.css

## 📁 Estructura del proyecto
```
├── index.html
├── css/
│   ├── normalize.css
│   └── styles.css
└── Img/
    └── hero.jpg
```

## ✨ Características
- Diseño responsive (mobile first)
- Navegación adaptable
- Sección hero con imagen de fondo
- Grid de servicios en 3 columnas
- Formulario de contacto
- Variables CSS para manejo de colores
